# Cineman-a---plus- # Cine Manía Plus

Bienvenido a **Cine Manía Plus**, una plataforma visual donde compartimos y promocionamos contenido audiovisual original, especialmente diseñado para redes sociales como Facebook.

## ¿Qué encontrarás aquí?
- Espacios para insertar y mostrar tus videos de Facebook
- Diseño adaptable (responsive) listo para móviles
- Enlaces para compartir contenido fácilmente
- Preparado para monetización social

## ¿Cómo usar este sitio?
1. Sube este repositorio a tu cuenta de GitHub
2. Activa GitHub Pages desde la sección **Settings → Pages**
3. Inserta tus enlaces de video en el código donde está marcado
4. Comparte tu enlace `https://TU_USUARIO.github.io/cinemania-plus` en tus redes
